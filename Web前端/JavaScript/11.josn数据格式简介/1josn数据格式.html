1.JSON(JavaScript Object Notation，js对象标记法)是一种轻量级的数据交换格式
2.JSON基于ECMAScript规范的一个子集，采用完全独立于编程语言的文本格式来存储和表示数据
3.JSON字符集必须是UTF-8，JSON的字符串规定必须使用双引号""，键名也必须用双引号""
4.任何支持的数据类型都可以通过JSON来表示
5.JavaScript内置了JSON的解析
6.JSON本质是字符串
7.JSON和js对象转换
a.js对象转JSON
（1）var json0=JSON.stringify(obj);
（2）第二个参数（可选）传入数组或函数
传入数组转换数组指定属性
传入函数调用该函数，并传入每个成员的键和值（给定参数应该有两个参数）
（3）第三个参数（可选）文本缩进、空格、换行符（没有第二个参数可以赋null）
b.JSON转js对象
（1）var js=JSON.parse(json);
（2）第二个参数（可选）传入传入一个函数
函数应有两个参数键和值
