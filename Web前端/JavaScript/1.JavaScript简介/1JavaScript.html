1.什么是JavaScript
a.Java和JavaScript（以下简称js）没有关系
b.一种嵌入html的脚本语言
2.js特点
a.脚本语言，通过小程序段的方式运行
b.解释语言，运行时逐行解释
c.面向对象语言
d.安全性语言
a.具有Web安全特性
b.不允许访问本地的磁盘
c.不允许对网络文档进行修改和删除
d.只能通过浏览器实现信息浏览或动态交互
e.跨平台性
js的执行依赖环境是浏览器而非os
3.ES、JS、JScript
a.ECMA国际标准化组织
b.ECMAScript简称ES，可以理解为js的一个标准，由国际标准化组织制定
c.JScript微软公司开发的一种脚本语言，起源是网景公司和微软公司的浏览器之争
d.JScript和js的差别微乎其微
4.script引入和注释
a.script标签内部引入
b.同css一样也可以行内引入，行内引入需要结合事件才可以引入
c.注释
（1）单行注释//
（2）多行数值/**/
5.输入输出工具
a.prompt()输入弹窗
b.alert()输出弹窗
会阻塞程序执行
c.document.write()输出到页面
d.console.log向控制台输出信息
不会阻塞程序执行
6.引入区域和执行顺序
a.js代码可以引入在html的任何区域
b.js代码执行时必须等到之前的代码执行后才能执行，与引入方式无关
7.js的阻塞性
a.js对DOM树的阻塞
（1）DOM（Document Object Model）文档对象模型
（2）当DOM树没有加载完成时，js可能会出错
b.解决的办法
（1）将js代码放在文档底部
（2）添加window.onload=function(){}事件
8.js发生错误时的阻塞
a.js发生错误时下面的代码不会执行
b.可以根据浏览器控制台中的错误提示进行差错
c.可以用弹窗等输出方式差错
