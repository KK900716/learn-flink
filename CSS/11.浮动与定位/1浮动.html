1.浮动
为了让块级元素不独占一行，可以使用浮动，让块级元素呆在一行
2.浮动元素的特点
a.不设置浮动时是标准的文档流
b.浮动元素脱离文档流，后面的元素会自动补位，会把不浮动的元素盖住
c.浮动元素会被父元素宽度约束
d.当浮动元素上一行是正常这个浮动只能呆在当前行
（1）left左浮动 元素会从上一行最右出来向左跑，直到遇到浮动属性元素停止
（2）right右浮动 元素会从上一行最左出来向右跑，直到遇到浮动属性元素停止

4.清除浮动
a.clear:left;清除元素左侧浮动
b.规定元素一侧不能有浮动
c.清除浮动时，要考虑页面加载顺序，代码会按顺序执行，不会影响到本元素，会影响后继元素
5.父元素高度塌陷
a.由于父元素的子元素设置了浮动，会导致父元素高度塌陷
b.解决办法
（1）设置父元素高度
（2）给最后一个元素清除浮动元素
（3）可以使用伪元素来设置第二个
ul::after{ content: ""; display:table; clear: both; } （4）父元素设置有高度时不用清除，父元素没有官渡时不用清除
