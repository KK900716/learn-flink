1.html用来搭建网页结构
2.css用来给网页添加样式
3.css的优点
撸起袖子加油干
a.丰富的样式
b.样式和结构分立，便于维护和管理
c.减少代码量、加快代码加载速度
d.代码的复用性强
e.多个页面可以使用同一个css样式表
4.css的缺点
a.浏览器对css支持程度不同
5.css书写格式
a.属性值是对属性的描述
b.属性名必须是一个合法标识符
c.文字颜色
color
d.文字大小
font-size
e.文字加粗
font-weight
f.背景颜色
backgroundcolor
g.宽高
width、height
6.css不区分大小写，但推荐使用小写
7.多条属性分割之间需要用;隔开
8.css引入方式有
a.行内样式
（1）将css代码写在元素开始标签的style属性中
（2）可以设置多个样式，使用;分割
b.内部样式
内部样式用到选择器，写在style标签之内，一般放在head标签内
c.外部样式
（1）引入外部的.css文件，外部样式同内部样式相同要用到选择器
（2）同一个页面可以引入多个css文件，同一个页面可以引入多个css文件
d.导入样式
（1）将一个css文件导入另一个css文件
（2）使用导入样式的加载会滞后html，所以这种样式不常用
@import url("b.css");
