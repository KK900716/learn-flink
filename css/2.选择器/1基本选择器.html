1.标签选择器
a.无论标签隐藏有多深，都可以选中
b.标签选择器选在的是所有是选择了一类
c.所有的标签都可以是选择器
div{ color: yellowgreen; }
2.ID选择器
a.使用id选择器时要在id前加#
b.id选择器只能选择一个元素，原因是id值唯一
c.id值以字母数字开头，后跟字母数字下划线，id不要与标签名重复
#idname{ color: violet; }
3.类选择器
a.使用类选择器要在类前加.
b.类选择器可以被很多个元素共同使用
c.同一个元素可以有多个类名
d.尽量在css中使用类选择器，而非id选择器，id选择器一般是js中使用
.classname{ color: aqua; }
4.通用选择器
a.通用选择器可以选中页面中的所有元素
b.使用通配符*进行选择
*{ background-color: yellow; }
