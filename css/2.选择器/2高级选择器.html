1.后代选择器
a.后代选择器描述的是一种祖先结构关系，并不一定是父子关系
b.祖先和后代之间要有一个空格隔开
c.后代之间的关系可以是任何基本选择器
.ynj .a2{ color: red; }

2.并集选择器
a.选择多个
b.选择时用,隔开
c.后代之间的关系可以是任何基本选择器
div,p{ background-color: aqua; }
3.交集选择器
a.选择交集
b.选择器间没有空格
c.后代之间的关系可以是任何基本选择器
d.交集选择器中一般没有id选择器
e.如果选择标签和类的交集，标签要放在类之前
div.abc{ background-color: yellow; }
