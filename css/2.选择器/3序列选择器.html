1.第一子元素选择器、最后子元素选择器
a.注意：选中的是该元素的父元素的第一个子元素（子元素要求必须是该元素），即同时满足第一子元素、和该元素两个条件才能被选中
b.最后子元素和第一子元素的使用方法相同
c.通常在列表中使用较多
div:first-child{ color: aquamarine; }
div:last-child{ color: brown; }
2.nth-child()选择器
a.选择的是该元素的父元素的第i个子元素
b.参数为第几个元素
（1）可以写常数
（2）可以写表达式
例如：（n从0开始直到遍历完整个内容数量） 2n+1表示奇数
2n表示偶数
-n+6表示前6个
n+4表示第4个以后的所有
（3）该选择器优先级高于高级选择器，因此可以使用交集选择器和并集选择器来补充选择内容
