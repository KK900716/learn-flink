1.ajax简介
a.Ajax即Asynchronous Javascript And XML（异步JavaScript和XML）
b.通过ajax可以在浏览器中向服务器发送异步骑牛，最大的优势：无刷新获取数据
c.ajax不是编程语言，而是一种将现有的标准组合在一起使用的新方式
2.XML简介
a.可扩展标记语言，标准通用标记语言的子集，简称XML。是一种用于标记电子文件使其具有结构性的标记语言
b.被设计用来传输和存储数据
c.xml中没有预定义标签，全部是自定义标签
3.JSON
JSON格式已经取代了xml格式
4.ajax优点
a.可以无刷新页面而与服务器进行通信
b.允许你根据用户事件来更新部分页面内容
5.ajax缺点
a.没有浏览历史，不能回退
b.存在跨域问题（同源）
c.SEO不友好（搜索引擎优化）
