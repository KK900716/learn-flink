	3. 权限管理命令
		1. chmod
			1. chmod命令
				1. 英文：change the permissions mode of a file
				2. 命令所在路径：/bin/chmod
				3. 执行权限：所有用户
				4. 功能：更改文件权限
			2. 语法：chmod [{ugoa}{+-=}{rwx}] [文佳或目录] [mode=421] [文件或目录] -R [递归修改]
				1. {ugoa}：所有者，所属组，其他人，所有人
				2. {+-=}：添加，删除，替换
				3. {rwx}：读，写，执行 ,分割可以同时设置多个权限
				4. mode=421：指定权限 r-4 w-2 x-1 即7：rwx 6：rw 1：r
				5. -R：递归更改
			3. 权限解释
				1. r 读权限 可以查看文件内容 可以列出目录中的内容
				2. w 写权限 可以修改文件内容 可以在目录中创建、删除文件
				3. x 可以执行文件 可以进入目录
		2. chown
